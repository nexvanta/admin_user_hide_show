# admin_user_hide_show

Take full control of user visibility in your Odoo system. This module allows administrators to selectively hide or show user profiles and admin accounts from standard views, ensuring a cleaner and more secure user management experience.

* **Profile Visibility Toggle:** Easily hide or show specific Admin or User profiles from the main Users list.
* **Role-Based Access:** Restrict who can see hidden users based on advanced security groups.
* **Security Focused:** Prevents sensitive administrative accounts from being visible to unauthorized internal staff.
* **Seamless Integration:** Fully compatible with Odoo's native Settings and User Management menus.
* **Stored Searchability:** Hidden status is stored in the database, allowing for easy filtering in developer mode.
